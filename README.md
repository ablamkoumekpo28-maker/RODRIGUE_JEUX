## 📥 Installation (Pour les joueurs)
1. Téléchargez la dernière version dans l'onglet **[Releases](https://github.com/votre_nom_utilisateur/nom_du_repo/releases)**.
2. Décompressez le fichier `.zip`.
3. Lancez `import pygame.exe` (ou renommez-le selon votre préférence).

## 💻 Développement (Pour les contributeurs)
Si vous souhaitez modifier le code :
1. Clonez ce dépôt : `git clone https://github.com/votre_nom_utilisateur/nom_du_repo.git`
2. Installez les dépendances : `pip install pygame`
3. Lancez le jeu : `python main.py`
